# Reports

`/[locale]/reports`

**Purpose:** Cross-cutting analytics.

## Tabs / sub-pages

| Tab | Path | What's there |
|---|---|---|
| Sales | `/reports/sales` | Revenue by channel/period, top products, top clients |
| Production | `/reports/production` | Throughput (cuts/week), banca on-time rate, scrap % |
| Inventory | `/reports/inventory` | Stock value over time, fabric utilization, slow movers |
| Costs | `/reports/costs` | Unit cost trend (spec + print + labor), margin per SKU |

Charts are server-rendered where possible. Each tab has a date-range picker and CSV export.
